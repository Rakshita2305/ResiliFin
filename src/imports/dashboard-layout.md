Layout

Create a clean 3‑section dashboard layout:

1. Scenario Selection Panel (Left Side)A card-based or list-based selector where the user can choose one simulation scenario.

Include the following selectable scenarios with icons:

Job Loss

Salary Reduction

Medical Emergency

Expense Inflation

Loan Burden

Chained / Compound Shock

Custom Event

Each scenario card should include:

icon

title

short description

“Run Simulation” button

2. Scenario Configuration Panel (Center)

When a user selects a scenario, show adjustable parameters such as:

Job Loss

Income becomes 0

Display monthly expenses

Show savings coverage estimate

Salary Reduction

Slider for income reduction (default 70%)

Display recalculated cash flow

Medical Emergency

Input field for medical cost

Option for random emergency cost

Expense Inflation

Slider for inflation percentage (default 20%)

Loan Burden

EMI input field

Loan duration selector

Compound ShockAllow user to combine multiple events such as:

Job loss + medical emergency

Salary reduction + inflation

Custom Event

User defines event name

Input financial impact amount

Include Run Simulation button at the bottom.

3. Simulation Results Dashboard (Right Side)

Display the results visually with the following components:

Financial Survival Timeline

Line chart showing savings balance over time (months)

Survival Months Indicator

Large metric card displaying“Your savings will last: X months”

Cash Flow Breakdown

Income vs Expenses bar chart

Risk Indicator

Color coded indicator

Green = Stable

Yellow = Moderate Risk

Red = High Risk

Insights PanelAuto-generated insights such as:

“Savings will be depleted in 8 months after job loss.”

“Reducing expenses by 15% extends survival to 12 months.”

Timeline Controls

Add simulation horizon selector:

12 months

24 months

36 months

Display month‑by‑month projection.

Design Style

Use a modern fintech dashboard style:

light background

rounded cards

minimal icons

clear charts

responsive layout

data visualization focused

Color palette:

Blue for financial stability
